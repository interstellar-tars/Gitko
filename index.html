Gitko Analytics Tracking Guide

Heads up! You are currently viewing the development version. Production and beta are not yet available.

🚀 Quick Start

To start tracking your website traffic with Gitko, simply add the following script tag to the <head> of your HTML page:

<script src="https://gitko.pages.dev/track.js" defer></script>

📦 What Gets Tracked?

  • URL Path: The path of the page being visited
  • Referrer: The previous site that sent the visitor
  • User Agent: The browser & device information
  • Device Type: mobile or desktop based on screen width

🔐 Privacy & Security

Gitko does not collect any personal data or use cookies. Only anonymous traffic metadata is logged.

Tracking requests are validated against allowed origins, and the data is stored securely.

📊 Viewing Stats

To view analytics, access your Gitko Dashboard. It shows:

  • Pageviews per URL
  • Top Referrers
  • Device type distribution
