Gitko Analytics Tracking Guide

🚀 Quick Start

Add the following tag in the <head> of your website to enable Gitko tracking:

<script src="https://gitko.cubiodojo.workers.dev/track.js" defer></script>

📦 What Gets Tracked?

  • URL Path: The path of the visited page
  • Referrer: The referring URL
  • User Agent: Browser/device info
  • Device Type: mobile or desktop

🔐 Privacy & Security

No personal data or cookies are collected. Data is anonymous and secure.

Only requests from allowed origins are accepted.

📊 Viewing Stats

Visit the Gitko Dashboard to see your analytics:

  • Pageviews by URL
  • Top Referrers
  • Device distribution
